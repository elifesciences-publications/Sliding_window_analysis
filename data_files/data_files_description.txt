The file "RPKM_compiled_slidingWindow.txt" contains the estimated RPKM values for each sliding window from reads in https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE106171 and reads from https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE106168. The file is tab-delimited.

The Rockhopper_output_file should also be placed in this folder for reference. This file is available on NCBI GEO https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE106171. called "NC_999999_transcripts.txt"
